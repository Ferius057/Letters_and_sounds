# Letters_and_sounds
Подсчет букв и звуков в русском слове. 

## Requirements:
- Java

----------------------------------------------------------------------------------------------

## Help:
- Программа не учитывает регистр и пробелы.

## Usage:
#### Linux:
```
# Установка репозитория
$ git clone https://github.com/Ferius057/Creating-Posts_VK.git

# Устоновка прав
$ chmod +x start.sh

$ Запуск
$ ./start.sh
```

```

```
